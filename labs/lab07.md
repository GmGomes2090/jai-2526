### 3.1. GUI, Markers, Indicators and Interfaces

Open EXAMPLE_1_GUI and check:

- Move boat (WASD)
- Fishing Markers (what happens when the marker leaves the screen?)
- Interact with the marker
- 'E' for overrlay interface
- Arrow for HUD inventory example

### 3.2. Shader Graph

Open EXAMPLE_2_SHADER. Open the WaterShader_V0 and check:

- Where do I change the water texture
- How can I change the color of the ocean?
- How do I create a material for the shader?

Switch the water material for WaterShader_V1 and open the respective material:

- Compare it to WaterShader_V0 and see what changed, and how it affected the visuals of the scene.
- Try to play with the input parameters.

Open Moss Shader and WaterShader_V2 to see more examples:

- How do you do you use sample Textures?
- Check the triplanar example to see a texture that is mapped in every axis.
- Left click on a node to “Open Documentation”.
- Repeat the process that you did for WaterShaderV1.
- In the Water shader, check the WS_Gerstner sub-shader.

### 3.3. Procedural terrain generation - Noise

Open EXAMPLE_3_ProceduralTerrainWNoise:

-  Try to understand the World Mesh Generator Script and the TerrainHeightShader.
- Check the InspectorButton script for an example of a script that changes the Unity UI
(adds two buttons to the World Mesh Generator Script's interface).
- Change parameters in TerrainHeightMap.mat
- Generate texture in World Mesh Generator Script (in MapGen gameObject in the
scene).
- Reopen Assets
- Add texture again, and press Generate.

### 3.4. Bézier Curves

Open EXAMPLE_4_BezierCurves and check the `Follower` and `BezierCurve` game objects. This example shows a boat that follows a complex 3D curve composed of several Bézier Curves:

- Try to edit the control points of the curve and see how it affects te boat's trajectory.
- Check the BezierCurve node in the Scene.
- Study the Follower script.

Check the [this tutorial (catlikecoding.com)](https://catlikecoding.com/unity/tutorials/curves-and-splines/)


### 3.5. Post-Processing

Open EXAMPLE_5_PostProcessing:
- Check the `PostProcessing` game object, play around with its values a see how they affect the scene. 
- Open the `LinearDepthFog.shadergraph` and try to invert the colors on the screen

**Note**:  if you don’t see fog in the game scene and if you have a warning on your console,
you need to open the main camera game object and click the Switch the Renderer Asset,
this will open a menu; on the Renderer asset step drag and drop the Ultra_PipelineAsset_ForwardRenderer.

In the Post Processing example:

- can you see orange fog? Check the LinearDepthFog.shadergraph and the Ultra_PipelineAsset_ForwardRenderer.
- Create a default PostProcessing effect with:

    - Create Volume>Global Volume node
    - Profile New
    - Add override, choose an effect. Ex: Film Grain.

Check [this tutorial](https://docs.unity3d.com/Packages/com.unity.render-pipelines.universal@15.0/manual/containers/post-processing-custom-effect-low-code.html)
